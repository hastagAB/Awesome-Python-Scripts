# IP Address 

This scripts will give you some useful information about your ip address. For instance:
- IP address 🧭 
-    Continent 🌍
-    Country 🗾
-    City 🏘️
-    Area 🏠
-    Postal code 📮
-    Isp 🏢 
-    Using Mobile 📱
-    Using Proxy 🌐

## Installation
After downloading the script you can simply install requirement by:\
> `pip install -r requirements.txt`

and then 
> `python main.py`
